Gherkin toDoList

toDoList

Feature: add "Selenium" in to do list;

Scenario:

    Given: Be on the website "https://lambdatest.github.io/sample-todo-app/";
    When: Type in the "want to add more" bar: Selenium;
    And: After typing the Selenium task, click the "add" button;
    Then: Website add the Selenium task in to do list;

toDoList10itens

Feature: add 10 tasks in to do list;

Scenario:

    Given: Be on the website "https://lambdatest.github.io/sample-todo-app/";
    When: Type in the "want to add more" bar the task to be added, then press the "add" button and type the other task;
    And: After typing the task, click the "add" button, then comeback to the bar and type another task;
    Then: Website add the task in to do list;
